Privacy Policy — NBA GBQ API & Custom GPT

Effective: 2025-10-02

This Custom GPT (“Service”) calls the NBA GBQ API hosted at https://nba-gbq-api-896368614747.us-central1.run.app to fetch NBA statistics and z-scores.

What data is processed

  • User prompts you enter into the GPT (e.g., dates, player IDs).
  • HTTP request metadata logged by the API platform (e.g., timestamp, path, status code, and IP per your cloud logging settings).

How data is used

  • To generate answers (query BigQuery and return results).
  • To monitor reliability and abuse (basic request logging and error diagnostics).

Data sharing

  • Infrastructure providers (e.g., Google Cloud) process logs and metrics to operate the service.

Retention

  • API logs are retained according to the project’s Google Cloud Logging retention settings.
  • No separate database is kept for user prompts beyond normal logs.

Security

  • Transport encrypted via HTTPS.
  • Access to cloud resources restricted to project administrators.

Your choices

  • You may request deletion of logs that reasonably identify you, subject to technical limits of cloud logging.
